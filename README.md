# GAN-based-Old-Photo-Restoration
本项目围绕老照片修复这一实际应用需求，设计并实现了一种基于生成对抗网络（GAN）的图像修复系统。针对老照片中常见的模糊、破损、分辨率低以及黑白缺失颜色等问题，构建了一个多模块级联的修复流程，对图像进行逐步优化处理。  在方法上，系统采用“人脸修复—超分辨率增强—图像上色”的处理顺序。首先利用人脸修复模型对照片中的人物面部结构进行恢复，提升关键区域的细节表现；随后通过超分辨率模型对整体图像进行清晰度增强，使图像更加锐利；最后引入图像上色模型，将黑白图像转换为彩色图像，从而提升整体视觉效果。各模块之间通过统一的数据接口进行连接，形成完整的修复管线。
